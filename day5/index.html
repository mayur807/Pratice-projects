This HTML CHEATHEET is for You!

  1. 1. HTML Boilerplate

    This is the basic starting point of every HTML page. It tells the browser this is an HTML5 document.

    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body>
  <!-- Your content goes here -->
</body>
</html>

  2. 2. Text Align (Center Text)

    This centers the text inside its container.

    .center-text {
  text-align: center;
}

    Example:

    <p class="center-text">Hello! I'm in the center.</p>

  3. 3. Padding and Margin

    Padding = space inside the box
    Margin = space outside the box

    .box {
  padding: 20px;  /* space inside */
  margin: 10px;   /* space outside */
}

    Example:

    <div class="box">I have space around me!</div>

  4. 4. Font Size and Color

    Changes how big and colorful the text is.

    .text {
  font-size: 24px;
  color: red;
}

  5. 5. Background Color

    Changes the background of any element.

    .box {
  background-color: yellow;
}

  6. 6. Flexbox (for layout)

    Flexbox helps arrange items in a row or column and center them easily.

    .container {
  display: flex;
  justify-content: center; /* center left to right */
  align-items: center;     /* center top to bottom */
}

    Example:

    <div class="container">
  <div>Box 1</div>
  <div>Box 2</div>
</div>

  7. 7. Border

    Adds an outline around elements.

    .bordered {
  border: 2px solid black;
}

  8. 8. Hover Effect

    Change style when mouse moves over an element.

    .button:hover {
  background-color: green;
  color: white;
}

  9. 9. Width and Height

    Sets the size of elements.

    .box {
  width: 200px;
  height: 100px;
}

  10. 10. Box Shadow

    Gives a soft shadow behind the element to look 3D.

    .shadow {
  box-shadow: 2px 2px 8px gray;
}

  11. 11. Responsive Image

    Makes the image adjust to screen size.

    img {
  max-width: 100%;
  height: auto;
}

  12. 12. Overflow Scroll

    Adds scroll if content is too big for the box.

    .scroll-box {
  width: 200px;
  height: 100px;
  overflow: auto;
}

  13. 13. Display

    Changes how elements behave in layout.

    .none {
  display: none; /* hides element */
}
.block {
  display: block; /* shows full width */
}
.inline {
  display: inline; /* stays in line with other text */
}

  14. 14. Positioning (Relative + Absolute)

    Moves elements by using top, left, right, or bottom.

    .outer {
  position: relative;
}
.inner {
  position: absolute;
  top: 10px;
  left: 20px;
}

  15. 15. Media Query (Mobile Responsive)

    Changes styles on small screen devices like phones.

    @media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}
